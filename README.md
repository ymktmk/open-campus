## Set up

1. Drop to local 

```
git clone https://github.com/ymktmk/open-campus.git
```

```
cd open-campus
```

2. Set API_KEY & CLIENT_ID

3. Enable cors 

4. Launch the web server

```
python -m SimpleHTTPServer 8000
```

or 

```
python -m http.server 8000
```
